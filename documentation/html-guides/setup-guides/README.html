Power Platform Setup & Documentation

Complete setup guide for SharePoint Permission Scanner project and Power Platform development.

Quick Start

1. Install CLI Tools

# Run the installation script
.\cli-tools\install-all.ps1



Verify installation

.\cli-tools\verify-installation.ps1

2. Authenticate

# Power Platform CLI
pac auth create



Microsoft 365 CLI (requires Azure AD app - see cli-tools/m365-cli.md)

m365 login



Azure CLI

az login

3. Select Environment

# List environments
pac env list



Select environment

pac env select --environment "Default-..."
