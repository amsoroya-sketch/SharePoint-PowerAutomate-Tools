Dataverse CLI Reference

Commands for managing Dataverse tables and data via CLI.

PAC CLI Commands

Authentication

# Create auth profile
pac auth create



List profiles

pac auth list



Select profile

pac auth select --index 1



Check connection

pac env who

Environment

# List environments
pac env list



Select environment

pac env select --environment "Environment Name"



Get environment details

pac env who

Table Operations

List Tables

pac table list

Get Table Schema

pac table get --table LOGICAL_NAME

Data Operations

Export Data

# Export with schema
pac data export --schemaFile schema.xml --dataFile data.zip



Export specific table

pac data export --table sp_scansession --dataFile sessions.zip

Import Data

pac data import --data data.zip

Solution Operations

Export Solution (includes tables)

pac solution export --name SolutionName --path solution.zip --overwrite

Import Solution

pac solution import --path solution.zip

Clone for Source Control

pac solution clone --name SolutionName --outputDirectory ./src

Project Tables

Schema Names

Scan Sessions:      sp_scansession
Folder Records:     sp_folderrecord
Permission Entries: sp_permissionentry

Common Commands

# Export project solution
pac solution export --name SharePointPermissionScanner --path solution.zip --overwrite



List tables in environment

pac table list

Power Platform CLI (Alternative)

Using Power Platform Admin API

# Via Azure CLI
az rest --method GET \
  --uri "https://api.powerplatform.com/environments/{envId}/tables" \
  --headers "Authorization=Bearer $(az account get-access-token --resource https://api.powerplatform.com --query accessToken -o tsv)"

Scripting Examples

PowerShell: Export Table Data

# Set environment
pac env select --environment "Development"



Export data

pac data export --table sp_scansession --dataFile scansessions.zip




Write-Host "Data exported to scansessions.zip"

Bash: List Tables

pac table list --output json | jq '.[] | {name: .logicalName, displayName: .displayName}'

OData Queries (REST API)

Base URL

https://[org].crm.dynamics.com/api/data/v9.2/

Query Tables

# Get all scan sessions
GET /api/data/v9.2/sp_scansessions



Filter by status

GET /api/data/v9.2/sp_scansessions?$filter=sp_status eq 'Completed'



Select specific columns

GET /api/data/v9.2/sp_scansessions?$select=sp_name,sp_status,sp_totalfolders

Using Azure CLI

TOKEN=$(az account get-access-token --resource https://[org].crm.dynamics.com --query accessToken -o tsv)




curl -X GET "https://[org].crm.dynamics.com/api/data/v9.2/sp_scansessions" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Accept: application/json"
