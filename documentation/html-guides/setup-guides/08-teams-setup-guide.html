Microsoft Teams Integration Guide

Complete guide for integrating Power Platform with Microsoft Teams.

Access Teams

Desktop App

Download: https://www.microsoft.com/microsoft-teams/download

Web App

https://teams.microsoft.com

Power Apps in Teams

Add Power Apps Tab

1. Open Teams channel 2. Click + to add tab 3. Search "Power Apps" 4. Select app to embed 5. Click Save

Create App in Teams

1. Click Apps in Teams sidebar 2. Search "Power Apps" 3. Click Power Apps > Create an app 4. Build app in Teams-embedded studio

Dataverse for Teams

  • Simplified Dataverse included free with Teams
  • Create tables directly in Teams
  • Limited to Teams context

Flows from Teams

CLI Commands

m365 CLI for Teams

# List teams
m365 teams team list



Get team details

m365 teams team get --id TEAM_ID



List channels

m365 teams channel list --teamId TEAM_ID



Send message

m365 teams message send --teamId TEAM_ID --channelId CHANNEL_ID --message "Hello from CLI"

Create Team

m365 teams team add --name "Project Team" --description "Team for project collaboration"

Add Member

m365 teams user add --teamId TEAM_ID --userId USER_ID --role Member

Webhooks

Incoming Webhook

1. Go to channel > ... > Connectors 2. Add Incoming Webhook 3. Configure name and icon 4. Copy webhook URL

Post via Webhook

curl -X POST "WEBHOOK_URL" \
  -H "Content-Type: application/json" \
  -d '{
    "@type": "MessageCard",
    "summary": "Scan Complete",
    "sections": [{
      "activityTitle": "SharePoint Scan Results",
      "facts": [
        { "name": "Folders", "value": "42" },
        { "name": "Status", "value": "Completed" }
      ]
    }]
  }'

Best Practices

Apps

  • Keep Teams apps simple
  • Use responsive design
  • Test on mobile
  • Handle offline scenarios

Flows

  • Don't over-notify users
  • Use adaptive cards for rich content
  • Batch notifications when possible
  • Respect quiet hours

Security

  • Use app permissions carefully
  • Review connector permissions
  • Audit Teams integrations
  • Follow data governance policies
