Azure Functions Core Tools

CLI for developing and testing Azure Functions locally.

Installation

Using npm

npm install -g azure-functions-core-tools@4

Using winget

winget install Microsoft.AzureFunctionsCoreTools

Verify Installation

func --version

Authentication

Azure Functions CLI uses Azure CLI for authentication: 

az login

Create New Function Project

Initialize Project

# Create new project
func init MyFunctionApp --worker-runtime dotnet



Available runtimes: dotnet, node, python, powershell, java, custom

Create Function

cd MyFunctionApp



HTTP trigger

func new --name MyHttpFunction --template "HTTP trigger"



Timer trigger

func new --name MyTimerFunction --template "Timer trigger"



Queue trigger

func new --name MyQueueFunction --template "Azure Queue Storage trigger"

List Templates

func templates list

Local Development

Start Local Server

func start

Start with Debugging

func start --verbose

Test HTTP Function

curl http://localhost:7071/api/MyHttpFunction

Deployment

Deploy to Azure

# Using Azure CLI
func azure functionapp publish FUNCTION_APP_NAME

Fetch App Settings

func azure functionapp fetch-app-settings FUNCTION_APP_NAME

Publish Settings

func azure functionapp publish FUNCTION_APP_NAME --publish-local-settings

Settings Management

local.settings.json Structure

{
  "IsEncrypted": false,
  "Values": {
    "AzureWebJobsStorage": "UseDevelopmentStorage=true",
    "FUNCTIONS_WORKER_RUNTIME": "dotnet"
  },
  "Host": {
    "CORS": "*"
  }
}

Encrypt Settings

func settings encrypt

Decrypt Settings

func settings decrypt

Durable Functions

Create Orchestration

func new --name MyOrchestrator --template "Durable Functions orchestrator"
func new --name MyActivity --template "Durable Functions activity"
func new --name MyHttpStart --template "Durable Functions HTTP starter"

Extensions

Install Extension Bundle

Add to host.json: 
{
  "version": "2.0",
  "extensionBundle": {
    "id": "Microsoft.Azure.Functions.ExtensionBundle",
    "version": "4.*, 5.0.0)"
  }
}

Install Specific Extension

func extensions install --package Microsoft.Azure.WebJobs.Extensions.Storage --version 5.0.0

Useful Commands

CommandDescription
func initCreate new project
func newCreate new function
func startRun locally
func host startStart host (alternative)
func azure functionapp publishDeploy to Azure
func settings addAdd local setting
func extensions installInstall extension

Troubleshooting

ErrorSolution
"No functions found"Check function.json exists
"Port 7071 in use"Use --port 7072
"Storage connection"Set AzureWebJobsStorage in local.settings.json
"Extension not found"Run func extensions install
