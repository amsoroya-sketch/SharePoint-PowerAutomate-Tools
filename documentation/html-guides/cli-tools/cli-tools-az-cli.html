Azure CLI

Azure CLI for managing Azure resources, Entra ID, and Azure DevOps.

Installation

Windows (winget)

winget install Microsoft.AzureCLI

Windows (MSI)

Download from: https://aka.ms/installazurecliwindows

macOS

brew install azure-cli

Linux

curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

Verify Installation

az version

Authentication

Interactive Login

# Opens browser for authentication
az login



Specify tenant

az login --tenant TENANT_ID

Device Code Login

az login --use-device-code

Service Principal Login

az login --service-principal --username APP_ID --password CLIENT_SECRET --tenant TENANT_ID

Check Current Account

az account show

List Subscriptions

az account list --output table

Set Active Subscription

az account set --subscription "Subscription Name"

Logout

az logout

App Registration Commands

List App Registrations

az ad app list --output table

Create App Registration

az ad app create --display-name "App Name" --sign-in-audience AzureADMyOrg

Get App Details

az ad app show --id APP_ID

Add API Permission

# Add Microsoft Graph User.Read
az ad app permission add --id APP_ID --api 00000003-0000-0000-c000-000000000000 --api-permissions e1fe6dd8-ba31-4d61-89e7-88639da4683d=Scope

Grant Admin Consent

az ad app permission admin-consent --id APP_ID

Create Client Secret

az ad app credential reset --id APP_ID --append

Entra ID (Azure AD) Commands

Users

# List users
az ad user list --output table



Get user

az ad user show --id user@domain.com



Create user

az ad user create --display-name "User Name" --user-principal-name user@domain.com --password "Password123!"

Groups

# List groups
az ad group list --output table



Get group

az ad group show --group "Group Name"



List group members

az ad group member list --group "Group Name" --output table



Add member to group

az ad group member add --group "Group Name" --member-id USER_ID

Service Principals

# List service principals
az ad sp list --output table



Get service principal

az ad sp show --id APP_ID



Create service principal for app

az ad sp create --id APP_ID

Azure DevOps Extension

Install Extension

az extension add --name azure-devops

Configure Defaults

az devops configure --defaults organization=https://dev.azure.com/myorg project=MyProject

Login to DevOps

az devops login

Pipelines

# List pipelines
az pipelines list --output table



Run pipeline

az pipelines run --name "Pipeline Name"



Get pipeline run

az pipelines runs show --id RUN_ID

Repos

# List repos
az repos list --output table



Create repo

az repos create --name "RepoName"

Resource Group Commands

# List resource groups
az group list --output table



Create resource group

az group create --name MyResourceGroup --location eastus



Delete resource group

az group delete --name MyResourceGroup

Key Vault Commands

# List key vaults
az keyvault list --output table



Create key vault

az keyvault create --name MyKeyVault --resource-group MyResourceGroup --location eastus



Set secret

az keyvault secret set --vault-name MyKeyVault --name SecretName --value "SecretValue"



Get secret

az keyvault secret show --vault-name MyKeyVault --name SecretName

Output Formats

# Table (default)
az account list --output table



JSON

az account list --output json



YAML

az account list --output yaml



TSV (tab-separated)

az account list --output tsv



JSONC (colored JSON)

az account list --output jsonc

Query with JMESPath

# Get specific property
az account show --query name



Get multiple properties

az account show --query "[name, id]"



Filter list

az ad user list --query "[?displayName=='John Doe']"



Formatted output

az account show --query "{Name:name, ID:id}" --output table

Configuration

View Config

az config get

Set Defaults

# Default resource group
az config set defaults.group=MyResourceGroup



Default location

az config set defaults.location=eastus

Enable Auto-Upgrade

az config set auto-upgrade.enable=yes

Useful Commands

Get Access Token

# For Microsoft Graph
az account get-access-token --resource https://graph.microsoft.com



For Power Platform

az account get-access-token --resource https://api.powerplatform.com

Find Resources

# Find all resources
az resource list --output table



Find by type

az resource list --resource-type Microsoft.Web/sites --output table

Troubleshooting

ErrorSolution
"Not logged in"Run az login
"Subscription not found"Run az account list and az account set
"Permission denied"Check Azure role assignments
"Extension not found"Run az extension add --name EXTENSION_NAME

Clear Cached Credentials

az account clear
az login

Update Azure CLI

az upgrade

Check Extensions

az extension list --output table
az extension update --name azure-devops
