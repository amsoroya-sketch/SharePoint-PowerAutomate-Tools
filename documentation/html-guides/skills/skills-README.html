Invoking Skills

Automatic Invocation

Skills auto-invoke based on task context:
  • "Create a canvas app" → powerapps skill
  • "Build a flow for SharePoint" → power-automate skill
  • "Design the architecture" → software-architect skill

Manual Invocation

Use slash commands: 
/orchestration - Coordinate multi-domain task
/powerapps - Power Apps development
/power-automate - Flow development
/dataverse - Dataverse development
/architecture - Architecture design
/software-architect - Enterprise architecture
/alm-devops - ALM and DevOps
/azure-integration - Azure integration

Adding New Skills

1. Create Skill Directory

mkdir -p Z:/.claude/skills/new-skill

2. Create SKILL.md

# New Skill Name




Description of the skill.



Capabilities


  • What it can do
    • 



When to Use

When to invoke this skill.

3. Add Templates (Optional)
