Best Practices

1. Be Specific

❌ "Help me with Power Platform"
✓ "Create a canvas app to display scan results from Dataverse"

2. Provide Context

❌ "Fix this error"
✓ "My flow has InvalidTemplate error on the Get_Root_Folder_Path action.
   The expression is trigger()['text']"

3. Use Domain Terms

❌ "Make a button that saves stuff"
✓ "Create a submit button that patches the record to Dataverse"

4. Reference Existing Work

"Update the SharePoint Permission Scanner flow to add error logging"
"Modify the sp_scansession table to add a new column"

Troubleshooting Skills

Skill Not Activating

1. Use explicit slash command: /power-automate 2. Include domain keywords in request 3. Check skill is defined in .claude/skills/

Wrong Skill Activated

1. Be more specific about the domain 2. Use explicit skill command 3. Clarify the technology stack

Skill Missing Information

1. Provide more context 2. Reference specific files 3. Include error messages/screenshots
